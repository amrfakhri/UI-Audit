# Privacy Policy — Audit.AI Plugin

Audit.AI (“the plugin”) respects your privacy and is designed to operate without collecting or storing personal data.

---

## 1. Data Processing

The plugin processes design-related data locally within your Figma environment. This may include:

- Exported images of selected Figma artboards  
- User-uploaded screenshots  
- Basic layout and metadata extracted from the design  

This data is used solely to perform UI and UX analysis.

---

## 2. Third-Party Services

To generate AI-powered audit results, the plugin sends data to third-party AI providers selected by the user, such as:

- OpenAI  
- Anthropic (Claude)  
- Google (Gemini)  
- OpenRouter  

All requests are made using the user’s own API key.

These providers process the data according to their own privacy policies and security practices.

---

## 3. Data Storage

The plugin does **not**:

- Store any user data  
- Persist design files or screenshots  
- Maintain any external database or backend  

All data is processed transiently and is not retained after the request is completed.

---

## 4. API Keys

API keys entered by the user are:

- Stored only within the plugin session or local plugin storage  
- Never transmitted except directly to the selected AI provider  
- Never shared with the developer  

Users are responsible for securing their own API keys.

---

## 5. No User Authentication

The plugin does not require user accounts, login, or authentication of any kind.

---

## 6. Security

The plugin does not operate its own backend infrastructure. All data processing occurs either locally or through trusted third-party AI providers.

If a security issue is identified, it can be reported to the developer and will be addressed promptly.

---

## 7. User Responsibility

By using the plugin, you acknowledge that:

- You choose which AI provider to use  
- You provide your own API key  
- You are responsible for reviewing the privacy practices of the selected provider
